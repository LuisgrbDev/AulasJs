Aula 05 - Trabalhando com Listas em JavaScript

Declaração de Listas

JavaScript oferece a capacidade de trabalhar com listas, uma estrutura que permite armazenar e manipular conjuntos de dados.

        
            // Declaração e inicialização de uma lista
            let frutas = ['Maçã', 'Banana', 'Uva'];

Acessando Elementos em uma Lista

Os elementos em uma lista podem ser acessados por meio de índices.

        
            // Acessando o primeiro elemento da lista
            let primeiraFruta = frutas[0];

            // Acessando o segundo elemento da lista
            let segundaFruta = frutas[1];

Manipulação de Strings como Listas

Em JavaScript, é possível tratar strings como listas e realizar diversas operações.

    
        // Declaração e inicialização de uma string
        let texto = "JavaScript";

        // Acessando caracteres da string como se fossem elementos de um array
        for (var i = 0; i < texto.length; i++) {
            console.log("Caractere na posição " + i + ": " + texto[i]);
        }

        // Revertendo uma string
        let textoArray = texto.split('');
        let arrayRevertido = textoArray.reverse();
        let stringRevertida = arrayRevertido.join('');
        console.log("String Revertida: " + stringRevertida);
